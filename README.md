# docker-couchpotato

## Description

A Dockerfile for [Couchpotato](https://couchpota.to/).

## Volumes

### `/data`

Configuration files and state for Couchpotato.

### `/torrents`

Watch directory for torrents.

### `/downloads`

Incoming downloads directory.

### `/movies`

Movie library.

## Ports

### 5050

WebUI port.
